# End Task

End Task is exit of the Workflow Runtime.
:::caution

There should be only one End Task in the Workflow Definition.

:::

```json title="End Task Example"
{
  "id": "24a7188e-d17b-4a5f-94a0-89286edc8b9f",
  "name": "End Task",
  "type": "END",
  "params": {},
  "next": [],
  "previous": ["Test Task", "Wait for Me"]
}
```
